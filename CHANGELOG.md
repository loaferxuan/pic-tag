# Changelog

## [Unreleased]

### Changed

- 备份协议更新为 `pictag-data` `1.x.x`
- 导出文件命名更新为 `pictag-data-YYYYMMDD-HHmmss.json`
- 数据库结构初始化命名与主文档表述统一

## [1.0.0] - 2026-03-19

### Added

- 照片导入、标签管理、搜索筛选、统计分析和数据备份能力
- SQLite 本地数据存储与默认标签初始化流程
- 备份导入导出主链路与自动回填进度反馈
- Jest、ESLint 与 Prettier 工程基线
- 架构、备份格式、测试和贡献指南文档

### Changed

- 搜索页与照片详情页的标签分类区支持展开与收起
- 搜索页筛选头部显示当前摘要并保留展开状态
- 统计页“拍摄日期统计”区块支持展开与收起

### Removed

- 无
